MyMovies2 - Using Yii.js
========================
Ruby Version 1.9.3p448

Rails Version 3.2.9

To run MyMovies2
----------------
1. Open command prompt/terminal and change directory to the MyMovies2 folder.
2. Then executes the following commands:
       
       ```
       > bundle update

       > bundle install
       
       > rails s
       ```
       
3. Go to http://localhost:3000/
